Manual HTML, CSS y jQuery

HTML

Etiqueta html

La etiqueta html representa la raíz de un documento HTML. La etiqueta html es el contenedor de todos los demás elementos HTML.

Etiqueta head

El elemento head es un contenedor de metadatos (datos sobre datos) y se coloca entre la etiqueta html y la etiqueta del cuerpo. Los metadatos son datos sobre el documento HTML. No se muestran los metadatos. Los metadatos suelen definir el título del documento, el conjunto de caracteres, los estilos, los scripts y otra metainformación.

Etiqueta h1-h6

Las etiquetas h1 a h6 se utilizan para definir encabezados HTML. h1 define el encabezado más importante. h6 define el encabezado menos importante. Nota: Utilice solo un h1 por página; esto debe representar el encabezado / tema principal de toda la página. Además, no omita los niveles de encabezado: comience con h1, luego use h2, y así sucesivamente.

CSS

align-content

La propiedad align-content modifica el comportamiento de la propiedad flex-wrap. Es similar a align-items, pero en lugar de alinear elementos flexibles, alinea líneas flexibles.

animation

La propiedad de animación es una propiedad abreviada para:

  • nombre-animación
  • duración de la animación
  • función de temporización de animación
  • retardo de animación
  • recuento de iteraciones de animación
  • dirección de animación
  • modo de relleno de animación
  • estado de juego de animación

background

The background property is a shorthand property for:

  • background-color
  • background-image
  • background-position
  • background-size
  • background-repeat
  • background-origin
  • background-clip
  • background-attachment

jQuery

Hide - Show

El parámetro de velocidad opcional especifica la velocidad de ocultar / mostrar, y puede tomar los siguientes valores: "lento", "rápido" o milisegundos. El parámetro opcional de devolución de llamada es una función que se ejecutará después de que se complete el método hide () o show () (aprenderá más sobre las funciones de devolución de llamada en un capítulo posterior).

Fade

El método jQuery fadeIn () se utiliza para fundir un elemento oculto. El parámetro de velocidad opcional especifica la duración del efecto. Puede tomar los siguientes valores: "lento", "rápido" o milisegundos. El parámetro de devolución de llamada opcional es una función que se ejecutará después de que se complete el desvanecimiento.

Slide

El método jQuery slideDown () se usa para deslizar hacia abajo un elemento. El parámetro de velocidad opcional especifica la duración del efecto. Puede tomar los siguientes valores: "lento", "rápido" o milisegundos. El parámetro de devolución de llamada opcional es una función que se ejecutará después de que se complete el deslizamiento.
